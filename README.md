# PKAI Project Management

How Pete Kaminski works on projects with Claude Code.

This is a small companion wiki to the rest of the PKAI series. The other repos in the series cover individual tools — Obsidian, Git, Claude Code itself, the integrated stack. This one covers the **practice** of using them together to actually get work done.

The bet of this stack is that working with agentic AI is mostly project management. The model does the work; you organize it. Claude Code is powerful enough to make a mess if you don't keep it on rails — and structured enough to be transformatively useful if you do.

## Who this is for

People who already have Claude Code, Obsidian, and Git working, and want to know what to *do* with them. If any of those tools are still new, start with:

- [pkai-getting-started](https://github.com/peterkaminski-ai/pkai-getting-started) — install everything and get to a working setup.
- [pkai-obsidian-reference](https://github.com/peterkaminski-ai/pkai-obsidian-reference) — Obsidian on its own.
- [pkai-git-guide](https://github.com/peterkaminski-ai/pkai-git-guide) — Git for non-developers.

Once you have the tools, come back here.
